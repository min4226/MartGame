게임 매니저 인스턴트를 들어간 뒤에
GameManager.Instance
GameManager.Instance.Input => 이렇게 매니저를 각자 찾아줄 수 있으

일시정지 / 해제
GameManager.Pause()
GameManager.Unpause()

오브젝트 등록하기
GameManager.OnInitializeManager += 함수
GameManager.OnInitializeController += 함수
GameManager.OnInitializeCharacter += 함수
GameManager.OnInitializeObject += 함수

업데이트 해달라고 하기
GameManager.OnUpdateManager += 함수
GameManager.OnUpdateController += 함수
GameManager.OnUpdateCharacter += 함수
GameManager.OnUpdateObject += 함수

부숴지는 거 등록하기
GameManager.OnDestroyManager += 함수
GameManager.OnDestroyController += 함수
GameManager.OnDestroyCharacter += 함수
GameManager.OnDestroyObject += 함수

파일 가져오기
DataManager.LoadDataFile<T>(이름)
DataManager.TryLoadDataFile<T>(이름, out 파일받아갈변수)

게임 중간에 로딩을 추가로 해야 할 때 -> 초반에 로딩이 너무 길면
DataManager.LoadAllFromAssetBundle<T>(태그, 로드되면 할 일)

원하는 타입의 UI를 갖고싶다
UIManager.ClaimGetUI(타입)

열고싶다
UIManager.ClaimOpenUI(타입)

닫고싶다
UIManager.ClaimCloseUI(타입)

여닫고싶다
UIManager.ClaimToggleUI(타입)

팝업창 열렸으면 좋겠다
UIManager.ClaimPopUp(제목, 내용, 확인버튼)

에러창 열렸으면 좋겠다
UIManager.ClaimErrorMessage(내용)

이름만 쓰면 그 이름에 해당하는 오브젝트를 만들어준다! + 오브젝트 풀이면 풀링도 해줌
ObjectManager.CreateObject(이름)
대상을 파괴하는데 + 풀이면 풀에 넣어줌!
ObjectManager.DestroyObject(대상)

게임하는 중간에 로딩이 되어서 새로운 풀이 필요할 경우
ObjectManager.RegistrationPool(이름)

GameManager.Instance.Camera.SetMainCamera        : 이 카메라로 할래!
GameManager.Instance.Camera.GetRaycastResult2D    : 2D에서 화면이 가리키는 대상 가져오기
GameManager.Instance.Camera.GetRaycastResult3D    : 3D에서 화면이 가리키는 대상 가져오기

InputManager.OnMouseRightDown    += 입력받으면 할 일
